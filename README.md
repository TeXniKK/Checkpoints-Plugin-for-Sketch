# Installation

### Manuel:
Double click the `Checkpoints.sketchplugin` file to install.

### Sketch Toolbox:
1. [Download the Sketch Toolbox plugin manager app](http://sketchtoolbox.com) and install.
2. Search `Checkpoints ` in the app and install.
